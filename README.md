# Phishing URL Detector

A React-based web application that allows users to paste a URL and get an instant prediction from a trained Machine Learning (ML) model to determine whether the website is safe or a potential phishing site.  
This project demonstrates the integration of frontend web technologies with ML to enhance online safety.
